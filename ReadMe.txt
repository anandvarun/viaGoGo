To Run the program.

1) Go to the directory where you have downloaded the file via terminal.
2) Execute the command java EventsAround

Input File.

1)EventsInput.txt contains the 100 inputs
2)The first value contains the x value.
3)The second value contains the y value.
4)The third value contains the ticket prices.
5)The fourth value contains the quantity.
6)The fifth value contains the unique event Id.
